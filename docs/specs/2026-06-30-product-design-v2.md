# 家庭金融助手 V2 — 产品设计文档

> 状态：已确认 | 日期：2026-06-30
> 重构版本，替代 V1 设计文档

## 一、产品定位

个人银行资金管理中枢。核心场景：**银行资产提升活动追踪 + 北交所/港股打新管理 + 定时操作提醒**。

目标用户：1 人使用，管理 5-6 个银行账户 + 2-3 个券商账户。

## 三、模块设计

### 3.1 资产看板（首页）

进入应用后的默认页，展示当月资金全貌。

**活动概览卡片：**

每个已配置的银行活动一行，展示：
- 银行名称
- 上月日均（基准线）
- 目标日均
- 当前实际日均
- 差额（当前−目标）
- 达标状态：🟢已达标 / 🔴差XX万 / ⚪未设置目标

差额带进度条可视化。顶部汇总"本月总资产 vs 总目标"。

**快捷操作：**
- [新增活动] → 跳转活动配置
- 点击任意银行行 → 跳转该银行的台账列位置

### 3.2 资产台账

核心模块，Excel 式日×账户矩阵。

**月度视图：**

| 日期 | 建行WQ | 建行WQ2 | 交行 | ... | 总计 | 备注 |
|------|--------|---------|------|-----|------|------|
| 5月日均 | 97.1万 | 100 | 695万 | | | （基准行，灰色） |
| 目标日均 | 500万 | - | 800万 | | | （目标行，带*） |
| 6/1 一 | 10,000 | 0 | 699万 | | 700万 | |
| 6/2 二 | 970万* | 0 | 701万 | | 1671万 | |
| ... | | | | | | |
| 6/30 二 | 777万 | - | 702万 | | 1479万 | |
| **本月累计** | 1.50亿 | 3045万 | 2.41亿 | | | |
| **本月日均** | 500万 | 102万 | 805万 | | | |
| **资产提升** | 403万 | 101万 | 109万 | | | |
| **vs目标** | 100%✅ | - | 101%✅ | | | |

**录入方式：**

- **A. 直接填余额：** 点击单元格 → 输入数字 → 回车。和 Excel 操作习惯完全一致。
- **B. 记录转账：** 顶部快捷栏 `[从: 建行WQ ▼] → [到: 交行 ▼] [金额] [确认]`，自动更新两列当天余额，同时在备注列自动标注。

**智能特性：**
- 如果某天余额没有变动，单元格默认继承前一天的值（可手动清空）
- 目标日均在台账中用 `*` 前缀或浅色高亮标记
- 水平滚动支持多列
- 金额以万为单位显示（可切换精确模式）

### 3.3 活动管理

每月配置各银行资产提升活动。

**活动字段：**

| 字段 | 说明 | 必填 |
|------|------|------|
| 银行账户 | 下拉选择已配置的账户 | ✅ |
| 活动月份 | 如 2026年7月 | ✅ |
| 上月日均 | 基准线，用于计算资产提升 | ✅ |
| 目标日均 | 需达成的日均余额 | 可选 |
| 报名截止日 | 活动报名截止 | 可选 |
| 活动起止 | 活动有效日期范围 | ✅ |
| 达标条件 | 日均≥目标 / 月末时点≥目标 / 两者均需 | 可选 |
| 奖励金额 | 达标后的奖励（用于收益统计） | 可选 |
| 备注 | 活动名称等 | 可选 |

**活动列表：** 按月筛选，展示每项活动的目标/实际/差额/状态。

### 3.4 IPO 管理

#### 北交所

- 抓取近期新股信息（公司、代码、发行价、一手金额、截止时间）
- 每只新股支持填写**多账户资金分配**（国信 XX 万、申万 XX 万）
- 确认分配后自动同步到台账备注
- 根据申购日期自动生成提醒

#### 港股

- 独立子页面，港币隔离
- 仅记录"是否已申购"状态
- 系统自动在截止前 1 小时（9:00）提醒检查

### 3.5 提醒中心

**系统预设规则（可开关）：**

| 提醒 | 触发时间 | 条件 |
|------|---------|------|
| 银证转账 | 8:20 | 有申购时启用 |
| 国信证券申购 | 8:40 | 有北交所新股 |
| 申万证券申购 | 8:50 | 有北交所新股 |
| 港股申购检查 | 9:00 | 有港股新股 |
| 港股暗盘卖出 | 16:05 | 有上市首日股票 |
| 月初资产检查 | 每月1号 9:00 | 始终启用 |
| 月末活动检查 | 每月最后一天 15:00 | 可开关 |

**自定义提醒：** 保留模板+自然语言输入（从 V1 继承）。

**推送通道：** PWA 浏览器通知 + PushPlus 微信推送（双通道，从 V1 继承）。

### 3.6 账户管理

从 V1 扩展，每个账户增加：

| 扩展字段 | 说明 |
|---------|------|
| 币种 | CNY / HKD |
| 转账方式 | U盾+自己操作 / 手机银行 / 需要去柜台 |
| 单日限额 | 数字（元） |
| 单笔限额 | 数字（元） |
| 操作时段 | 如"工作日9:00-17:00" |
| 备注 | 自由文本 |

这些字段仅作**操作时的提示参考**，系统不自动校验。在转账操作栏选择转出银行时，该银行的转账规则以卡片形式展示。

## 五、页面导航

```
底部 Tab：
  📊 看板     📈 台账     🏦 IPO     🔔 提醒     ⚙ 设置

看板 → 点击银行 → 跳到台账该列
台账 → 按月切换 → 自动汇总
IPO → 北交所（配资）/ 港股（申购状态）
提醒 → 预设规则 + 自定义
设置 → 账户+转账规则 / 活动管理 / 推送 / 数据
```
